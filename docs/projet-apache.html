Contexte

Installation et configuration de VirtualHosts, commandes d'administration et scripts d'automatisation pour déploiements.

Fichiers fournis

Installation rapide (exemple)

sudo cp config/apache/mon-site.conf /etc/apache2/sites-available/monsite.conf
sudo a2ensite monsite.conf
sudo systemctl reload apache2

Commandes utiles

sudo a2ensite monsite.conf
sudo a2dissite monsite.conf
sudo systemctl restart apache2
tail -f /var/log/apache2/error.log
